![Demo](https://imgur.com/ZEhyBut.png)

## NPM Scripts

有兩個預先寫好的指令：

### `npm run dev-server`

用 [json server](https://github.com/typicode/json-server) 模擬出假API，讓資料能在local端運行

### `npm start`

執行網頁。

## 操作步驟

```
1. 點擊漢堡選單 註冊/登入 帳號
2. 漢堡選單會多出一個Dashboard按鈕
3. 進去後可以發布文章 & 上傳檔案
4. 回到漢堡選單 News 點擊 Load More News
5. 文章新增成功！！
```
